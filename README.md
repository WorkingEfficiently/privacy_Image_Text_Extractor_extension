# Image Text Extractor - Chrome Extension

**Image Text Extractor** is a lightweight Chrome extension that allows users to extract text from images quickly and efficiently. Simply drag an image into the popup or use the upload button, and the text will be extracted using the OCR.space API. Users can then copy the text to the clipboard with a single click.

## Features

- Drag & drop images directly into the popup
- Upload image from your computer
- OCR text extraction powered by OCR.space API
- Copy extracted text to clipboard with one click
- Modern, space-themed popup interface

## How to Use

1. Install the extension in Chrome (Developer mode for testing)
2. Click the extension icon to open the popup
3. Drag an image into the box or click "Upload Image" to select a file
4. Wait for the OCR to process the image
5. Copy the extracted text using the "Copy" button

## Permissions

- **storage**: To save minimal user preferences (optional/future)
- **clipboardWrite**: To copy extracted text to the clipboard

## Privacy

This extension does **not collect any personal data**. Images are only temporarily sent to the OCR.space API for text extraction. No data is stored or shared.
