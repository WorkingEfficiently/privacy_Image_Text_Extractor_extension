Privacy Policy for Image Text Extractor

Last updated: February 2026

1. Information We Collect

Image Text Extractor does not collect any personal data from users. The extension only processes images that you personally drag or upload into the popup.

Images are temporarily sent to the OCR.space API for text extraction. Extracted text is displayed in the popup and can be copied to your clipboard. None of your images or extracted text is stored or shared by the extension.

2. Permissions

  • storage: Used to store minimal user preferences for optional future features.
  • clipboardWrite: Used to copy extracted text to your clipboard.
  • No host permissions are requested or required.

3. Data Retention

Images are only sent temporarily to the OCR API for processing. Extracted text exists only within the popup until you copy it or close the popup. No data is stored permanently.

4. Security

All communication with OCR.space is encrypted using HTTPS. The extension does not track or transmit any other information.
